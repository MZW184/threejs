Exploring My Digital Creations: A Blend of JavaScript and React.

Welcome to my project showcase! Below, you’ll find a curated collection of links and videos
each a testament to my coding journey. Some dance to the rhythm of JavaScript, while others waltz
gracefully with React. Together, they form a symphony of pixels and logic. So, grab your virtual
ticket, dim the lights, and let’s dive into this showcase. Whether you’re a fellow developer, an aspiring
coder, or just a curious soul, there’s something here for everyone. Ready? Lights up!

I built a movie search application using React and JavaScript. The app allows users to search
for movies the by title or just a word. It fetches real-time data from a movie database
API and displays movie details, including posters and the name of the movie.
The project taught me valuable skills in handling APIs, creating UI components,
and managing user interactions.
Tech Stack: React, JavaScript and Movie database API.
